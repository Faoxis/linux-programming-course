### Файлы и файловые системы
<hr>

#### Базовые функции работы с файлами
* `fid = open(имя, флаги)` - открытие файла, отдает дискриптор файла
* `write(fid, ...)`
* `read(fid, ...)` 
* `close(fid)`
* `creat` - создание
* `unlink` - удаления

Для просмоотра справочной информации можно воспользоваться командой `man 2 имя_функции`.

#### Работа с каталогами и путями
* Каталог - это файл особого вида, который хранит определенную мета-информацию о других файлах или каталогах. Каталог описывает входящие в него файлы и каталоги.
* Работа с каталогами в `Linux` схожа с работой с файлами.
* Основные функции:
    * `DIR* opendir(const char *name);` - открытие каталога по имени.
    * `DIR* fdopendir(int fd);` - открытие каталога по файловому дискриптору.
    * `int closedir(DIR *);` - закрытие каталога
    * Структура `DIR` располагается в библиотеке `dirent.h`, а его конкретная реализация зависит от платформы.
